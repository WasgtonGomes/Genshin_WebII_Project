Projeto Final - Programação Web II

Este projeto foi desenvolvido como atividade final da disciplina Programação Web II, com o objetivo de aplicar na prática os conceitos de desenvolvimento web aprendidos ao longo do semestre. A proposta inicial consistiu na criação de uma aplicação completa utilizando Node.js no back-end, integrando operações CRUD (Create, Read, Update e Delete) com banco de dados e consumo de APIs externas.

A aplicação foi construída empregando as seguintes tecnologias e ferramentas:

  • Node.js — Responsável pelo papel central do back-end, permitindo a execução de JavaScript no servidor, gerenciamento das rotas, conexão com o banco de dados e integração com a API externa.
  • HTML5, CSS3 e JavaScript — Tecnologias fundamentais para a construção da interface e interação com o usuário.
  • Bootstrap — Framework CSS utilizado para estilização e responsividade das páginas.
  • MySQL — Sistema de gerenciamento de banco de dados relacional para armazenamento e manipulação das informações cadastradas pelos usuários.
  • Axios — Biblioteca para realizar requisições HTTP e integração com a API pública do jogo Genshin Impact, trazendo dados externos em tempo real.

Entre as principais funcionalidades desenvolvidas estão:

  • Cadastro de personagens personalizados pelos usuários.
  • Validação dos dados inseridos no formulário de cadastro.
  • Apresentação dos dados armazenados em tabelas provenientes do banco de dados.
  • Consulta e exibição dinâmica de informações vindas da API externa.

O projeto foi apresentado oficialmente em 27/11/2023, atendendo ao requisito de demonstrar não apenas a interface e as funcionalidades implementadas, mas também os trechos de código que refletem as principais decisões de design e arquitetura da aplicação.
