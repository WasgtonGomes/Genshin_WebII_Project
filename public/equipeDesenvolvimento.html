Principais Atividades no Projeto
  • Back-end: Desenvolvimento em Node.js, criação das rotas e integração com o banco de dados.
  • Front-end: Construção das interfaces com HTML, CSS, JavaScript e Bootstrap.
  • Banco de Dados: Modelagem e implementação no MySQL, além da manipulação de dados com operações CRUD.
  • Integração com API: Consumo de dados da API do Genshin Impact utilizando Axios e apresentação dinâmica no site.
  • Validação e Usabilidade: Implementação de verificações nos formulários e organização da navegação no sistema.

Este projeto foi desenvolvido em 2023 como atividade final da disciplina de Programação Web II.
